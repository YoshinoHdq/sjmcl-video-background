## 简介

动态视频背景是一款 SJMCL 扩展，允许你上传 MP4 或 WebM 格式的视频作为启动器主页的动态背景。

可与 sjmcl-city-weather（城市天气粒子）、sjmcl-weather-effects（背景粒子特效）叠加使用，打造个性化启动器界面。

> 📌 同时开启背景粒子特效时，若要改变背景粒子特效需先在扩展卡片删除视频再调整。

## 功能特色

- 🎬 支持 MP4 / WebM 格式视频作为动态背景
- 🧩 可与天气粒子、粒子特效等背景类扩展叠加
- ⚙️ 在扩展设置页上传和管理视频

## 使用方法

1. 在 Releases 下载 org.sjmcl.video-background.sjmclx
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 在扩展设置页上传你的 MP4 或 WebM 视频

## 兼容性

- 最低启动器版本: 1.1.3
